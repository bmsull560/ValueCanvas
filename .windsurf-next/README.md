# Windsurf Rules for ValueCanvas

This directory contains Windsurf AI rules for the ValueCanvas project.

## Structure

- `rules/00-global.md` - Complete VOS global rules
- `rules/01-ai-agents.md` - AI Agent module standards
- `rules/02-tools.md` - Tool libraries standards
- `rules/03-orchestration.md` - Orchestration layer standards
- `rules/04-memory.md` - Memory management standards
- `rules/05-communication.md` - Agent communication standards
- `rules/06-frontend.md` - Frontend component standards
- `rules/07-backend.md` - Backend service standards

## After Reinstalling Windsurf

These rules will automatically be available when you clone the repository. Windsurf reads rules from `.windsurf-next/rules/`.
